# Задача
	Стример Destiny 2: The Final ShapeRichardRoe: У тебя есть интерфейс. Строка, справа от строки кнопка начать. Под строкой и кнопкой находится таблица, под таблицей прогресс бар. 
	В строку вставляется ссылка. По нажатию на кнопку прога должна запросить сайт по этой ссылке и извлечь из него все внутренние ссылки. 
	После чего эти ссылки помещаются в таблицу. В таблице есть колонки ссылка, заголовок сайта по этой ссылке и текущий статус запроса этой ссылки.

	Стример Destiny 2: The Final ShapeRichardRoe: Когда нажимают на первую кнопку, то вместе нее появляются две другие кнопки: пауза и сбросить.
